# Guide de contribution

## Bienvenue !
Ce projet documente les recherches sur les manifestations des Algériens à Paris le 17 octobre 1961. Nous accueillons les contributions respectueuses et scientifiquement rigoureuses.

## Comment contribuer ?

### 1. Pour une simple suggestion
- Ouvrez une **Issue** avec le template "Contribution au projet"
- Décrivez votre suggestion
- Nous en discuterons ensemble

### 2. Pour ajouter du contenu
1. **Forkez** le dépôt (bouton "Fork" en haut à droite)
2. **Clonez** votre fork sur votre ordinateur
3. **Créez une branche** : `git checkout -b ma-contribution`
4. **Ajoutez votre contenu** dans le dossier approprié
5. **Commitez** : `git commit -m "Description de ma contribution"`
6. **Poussez** : `git push origin ma-contribution`
7. **Ouvrez une Pull Request** sur GitHub
